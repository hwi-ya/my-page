## ✨ <mark style="background: #FFF3A3A6;">용어 정리</mark>
&nbsp;  **🧬 <mark style="background: #CACFD9A6;"> 개체(Individual)</mark>**<br>
  - 유전알고리즘에서의 의미 : 하나의 후보 해(Candidate solution)
  - 예시
    > 이진수 표현 : 101101<br>
    > 실수 표현 : [3.4, 2.1, -1.7]<br>
    > 순열 표현(여행자 문제) : [2, 5, 3, 1, 4]

&nbsp;  **🧬 <mark style="background: #CACFD9A6;"> 염색체(Chromosome)</mark>**<br>
  - 생물학적 의미 : 유전 물질을 담고 있는 하나의 집합
  - 유전알고리즘에서의 의미 : 하나의 후보 해를 표현하는 구조 (변수들의 조합)
  - 예시
    > 이진수 표현 : 101101<br>
    > 실수 표현 : [3.4, 2.1, -1.7]<br>
    > 순열 표현(여행자 문제) : [2, 5, 3, 1, 4]

&nbsp;  **🧬 <mark style="background: #CACFD9A6;"> 유전자 (Gene)</mark>**<br>
  - 염색체를 구성하는 요소 (개별 변수 값)
  - 염색체 내에서 특정한 위치(Locus)를 차지하며, 문제를 구성하는 변수(Parameter)와 연관됨.
  - 예시
    > 염색체 [A, B, C]에서 개별 요소 A, B, C가 유전자<br>
    > 이진수 염색체 101101에서 개별 숫자(0 또는 1)가 유전자<br>
    > 좌표를 표현하는 실수 염색체 [3.4, 2.1, -1.7]에서 개별 값이 유전자

&nbsp;  **🧬 <mark style="background: #CACFD9A6;"> 자손 (Offspring)</mark>**<br>
  - 특정 세대(time t)에 존재했던 염색체(부모)로부터 생성된 새로운 염색체
  - 교차(Crossover) 및 돌연변이(Mutation) 과정을 통해 부모와 유사한 유전 정보를 가짐
  - 새로운 세대를 구성하는 염색체들이며, 적합도가 높은 해를 만들기 위한 탐색 과정의 일부

&nbsp;  **🧬 <mark style="background: #CACFD9A6;"> 적합도 (fitness)</mark>**<br>
  - 개별 염색체(해)가 문제를 해결하는 데 얼마나 적합한지를 평가하는 값
  - 적합도 함수(Fitness Function)에 의해 계산되며, 함수는 일반적으로 최대화 또는 최소화 문제로 표현됨
  - 적합도가 높을수록 더 좋은 해이며, 선택 시 중요한 기준이 됨.

&nbsp;  **🧬 <mark style="background: #CACFD9A6;"> 유전 (Inheritance)</mark>**<br>
  - 부모의 형질이 자녀에게 전달되는 현상.
  - 유전 알고리즘에서는 교차(Crossover) 를 통해 부모의 유전자가 자손에게 전달됨.
  - 돌연변이(Mutation)가 함께 작용하여, 일부 유전자는 변화할 수도 있음.

## ✨ <mark style="background: #FFF3A3A6;">개체, 염색체, 유전자 간의 관계</mark>
- 예제 : 변수 개수가 2개인 최적화 문제
- 문제 : Minimize $f(x, y) = x^2 + y^2$
- 변수 : $x, y$ (두 개의 연속형 변수)
- 목표 : $f(x, y)$를 최소화하는 $(x, y)$ 조합을 찾기

<br>

- **<mark style="background: #CACFD9A6;">개체(Individual)</mark> = 하나의 후보 해(Candidate Solution)**
  > ✔ 개체는 **문제의 해(Solution)**를 나타냄  
  > ✔ 개체 하나는 $(x, y)$ 값의 조합을 가짐  
  > ✔ 즉, **"하나의 개체 = 하나의 $(x, y)$ 값"**  

  | 개체(Individual) | $(x, y)$ 값 |
  |------|--------------|
  | A    | $(3.2, -8.5)$ |
  | B    | $(-6.1, 2.9)$ |
  | C    | $(9.7, -1.3)$ |
  | D    | $(-2.4, 5.6)$ |

<br>

- **<mark style="background: #CACFD9A6;">염색체(Chromosome)</mark> = '개체'를 표현하는 유전 정보의 집합**
  > ✔ 염색체는 개체의 유전 정보를 담고 있는 구조  
  > ✔ $x$와 $y$ 값을 저장하는 실수형 리스트(벡터)로 표현 가능
  > ✔ 개체를 표현하는 정보(즉, 변수 값들의 조합)임

  | 개체(Individual) | 염색체(Chromosome) |
  |------|--------------|
  | A    | `[3.2, -8.5]` |
  | B    | `[-6.1, 2.9]` |
  | C    | `[9.7, -1.3]` |
  | D    | `[-2.4, 5.6]` |

<br>

- **유전자(Gene) = 개별 변수 값**
  > ✔ 유전자는 염색체를 구성하는 개별 요소  
  > ✔ 즉, **유전자는 변수 하나에 해당함**  
  > ✔ $x$와 $y$가 각각 하나의 유전자 역할을 함  
  > ✔ 염색체 `[x, y]`의 각 요소 = 하나의 유전자(Gene)
  > ✔ 유전자의 개수는 변수의 개수와 동일

  | 개체(Individual) | 염색체(Chromosome) | 유전자(Gene) - $x$ | 유전자(Gene) - $y$ |
  |------|--------------|------|------|
  | A    | `[3.2, -8.5]` | `3.2` | `-8.5` |
  | B    | `[-6.1, 2.9]` | `-6.1` | `2.9` |
  | C    | `[9.7, -1.3]` | `9.7` | `-1.3` |
  | D    | `[-2.4, 5.6]` | `-2.4` | `5.6` |

<br>

**✅ 최종 정리**
  > ✔ **개체(Individual) = 하나의 해 (하나의 $(x, y)$ 값 조합)**  
  > ✔ **염색체(Chromosome) = 개체를 표현하는 데이터 (즉, $[x, y]$ 리스트)**  
  > ✔ **유전자(Gene) = 염색체를 구성하는 개별 요소 (즉, 변수 값 $x$와 $y$)**

## ✨ <mark style="background: #FFF3A3A6;">유전알고리즘의 절차</mark>
&nbsp; 0️⃣ **초기 모집단(Initial Population) 생성**
 
&nbsp; 1️⃣ **적합도 평가(Fitness Evaluation)** 에 기반한 **부모해 선택(Selection)**
 
&nbsp; 2️⃣  **자식 해 생성(Offspring Generation)**
 - 부모해를 교차(Crossover)하여 새로운 개체 생성
 - 일정 확률로 돌연변이(Mutation) 적용
   
&nbsp; 3️⃣ **새로운 모집단 구성**(Next Generation Population)
 - 생성된 자식해들로 새로운 모집단을 형성
 
&nbsp; 🔁 **1~3 과정 반복**
 - 새로운 모집단을 다시 적합도 평가
 - 종료조건(세대 수, 적합도 기준 등)을 달성할 때 까지 반복하여 최적해를 탐색
